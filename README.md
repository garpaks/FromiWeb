# PaginaFromi3.0 - Neumática Fromi

> **Aviso:** Este proyecto ha sido **cancelado** por lo que se encuentra **inconcluso**.

## ¿Qué es este proyecto?
PaginaFromi3.0 es el desarrollo del sitio web informativo y corporativo para **Neumática Fromi**, un laboratorio de metrología con más de 14 magnitudes acreditadas ante la EMA (Entidad Mexicana de Acreditación). El propósito de la página era ofrecer una plataforma digital moderna para mostrar la historia del laboratorio, sus acreditaciones y captar clientes de industrias como la farmacéutica, alimenticia, automotriz, textil, eléctrica y siderúrgica.

## Funciones del Sitio
El sitio web cuenta con las siguientes secciones y funciones (algunas en estado base debido a la cancelación):
- **Inicio (Home):** Presentación del laboratorio, galería de clientes/industrias, línea del tiempo interactiva con la historia de la empresa y carrusel automático de frases célebres.
- **Servicios de Calibración:** Información detallada sobre las áreas acreditadas y el catálogo de servicios.
- **Certificados y Acreditaciones:** Espacio dedicado a mostrar el respaldo de la EMA y otras entidades reguladoras.
- **Verificentros:** Apartado específico para el servicio a centros de verificación vehicular.
- **Contacto y Cotizaciones:** Sección diseñada para que los clientes puedan solicitar cotizaciones y agendar programas de calibración en tiempo y forma.

## Tecnologías Utilizadas
Este proyecto está programado completamente con tecnologías frontend base (sin el uso de frameworks pesados):
- **HTML5:** Para la semántica y estructura de las múltiples vistas (`index.html`, `Contacto.html`, `Servicios_de_Calibración.html`, entre otras).
- **CSS3:** Para el diseño visual, estilos, layouts (`flexbox`/`grid`) y responsividad del sitio. Incluye el uso de fuentes externas (Montserrat).
- **JavaScript (Vanilla JS):** Para la lógica de interactividad en el cliente, como el cambio dinámico de información y temporizadores (por ejemplo, el carrusel de lemas en la página principal).
